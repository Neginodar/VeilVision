## Dataset

Input videos from the [INDRA — Indian Dataset for Road Crossing](https://www.kaggle.com/datasets/siddhi17/road-crossing-dataset):

- 104 videos, 26,000+ frames, 1.79 GB
- Pedestrian head-mounted camera POV
- Real Indian street traffic (Anand, Gujarat, 2019)
- Challenging: crowded scenes, helmeted riders, small distant faces

**Download:**
```bash
kaggle datasets download -d siddhi17/road-crossing-dataset
unzip road-crossing-dataset.zip -d Videos/
